### 什麼是 HTML？
* 英文：HyperText Markup Language
* 中文：超文字標記語言
* 用來描述網頁的一種語法標準
* HTML 不是程式語言(programming language)，是一種標記語言(markup language)
* HTML 使用標記標籤(tag)來描述網頁內容

### 一個空白的 HTML 範本

這是一個簡化的 HTML 範本

>    <html>
>    <head>
>      <title>這是網頁標題只能放文字</title>  
>    </head>
>    <body>
>      這裡放網頁內容
>    </body>
>    </html>
